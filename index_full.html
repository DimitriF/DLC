Collection of application made in R for HPTLC

Code for this website and all apps is on GitHub
rtlc
Application to explore HPTLC pictures with chemometrics.
tlc denoinsing
Artificial neural network on pixel patches. Denoising and feature extractions.
lipopeptidor
Brute force database of all possible combinaisons of lipopetide. Should facilitate the MS structural elucidation.
plate_prediction
Interactive predictive statistics made to work on a single plate, without preprocessing or variable selection.

Other Shiny apps that exist here

  • rtlc-ms - Application to link thin layer chromatography and mass spectrometry, really basic prototype.
  • tlcbrain - Application to explore the multidimensionality of tlc pictures.
  • tlc_var_selector - Application to select variables of interest in HPTLC pictures.
  • RBM classification - Deep learning on tlc video densitograms.
